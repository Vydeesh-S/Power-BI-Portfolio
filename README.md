# Power-BI-Portfolio
 Salesforce Case & SLA Insights Platform (Power BI)

Designed and developed an end-to-end analytics platform to consolidate Salesforce case data, SLA milestones, and operational metrics into a real-time performance monitoring system using Power BI. The solution enabled leadership and operations teams to monitor case trends, SLA adherence, and root causes of operational issues in a single unified dashboard.

 Key Contributions

* Built automated data pipelines integrating Salesforce Case and Milestone objects into Power BI for near real-time reporting.
* Developed interactive dashboards to track:

  * Case lifecycle trends (cases logged vs. closed)
  * SLA performance including Average Resolution Time (ART), violation rate, and backlog aging
  * Operational issue trends and monthly performance patterns
  * Customer-level issue patterns and top drivers
  * Case category contribution and workload distribution
* Implemented backlog segmentation (1–15, 15–30, 30–60, 60+ days) to highlight SLA risk zones and aging cases.
* Enabled deep-dive analysis through drill-downs for RCA by customer, error type, accountable team, and root cause narratives.
* Built advanced visualizations including hierarchical and Sankey analysis to map issue flows and root-cause relationships.
* Implemented data quality monitoring to assess partner/reseller reporting completeness and frequency.

Business Impact

* Improved SLA compliance by providing early visibility into aging backlogs and violation risks.
* Enhanced customer experience by identifying recurring issues and high-impact accounts.
* Eliminated manual reporting by automating monthly and quarterly operational reviews.
* Enabled data-driven resource planning and process improvements based on trend analysis and RCA insights.



These two additions make the project look like a real analytics platform rather than just a dashboard.
